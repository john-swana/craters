integrate

  • integrate(dt: number): void

  • Integrate physics using semi-implicit Euler method.

    Parameters

    • dt: number

      elapsed time in SECONDS. Velocities are therefore in px/s and accelerations in px/s² — which is what the internal tuning constants (maxSpeed = 4000 px/s, the resting-contact threshold) assume. With a fixed-step RenderLoop, pass loop.deltaSeconds (NOT loop.delta, which is milliseconds — that would scale every velocity by 1000×).

    Returns void
